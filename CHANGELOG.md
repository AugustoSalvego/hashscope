# Changelog

All notable changes to HashScope will be documented in this file.

## [1.0.0] - 2026-05-27

### Added

- Initial HashScope implementation.
- Interactive hash table laboratory.
- Separate Chaining mode.
- Open Addressing mode with Linear Probing.
- Fixed hash function: `Math.abs(key) % 10`.
- Memory board with 10 buckets.
- Collision visualization.
- Real-time hash preview before insertion.
- Load factor calculation.
- Multilingual support:
  - Portuguese
  - English
  - Italian
- Theory introduction section.
- Vercel deployment support.
- Project documentation.
